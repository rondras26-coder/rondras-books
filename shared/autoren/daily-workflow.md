# Daily Book Workflow

## Tagesablauf (Daily Roman)

| Zeit | Phase | Agent | Aufgabe |
|------|-------|-------|---------|
| 00:00 | 1 | AutoScribe | Neues Genre wählen, Konzept erstellen |
| 06:00 | 2 | AutoScribe | Kapitel 1-10 schreiben |
| 10:00 | 3 | AutoScribe | Kapitel 11-20 schreiben |
| 14:00 | 4 | AutoScribe | Kapitel 21-28 schreiben, Finale |
| 16:00 | 5 | Lektor | Lektorat & Feedback |
| 18:00 | 6 | Buch-Designer | Cover generieren |
| 20:00 | 7 | - | Fertig! |

---

##流程 (Pro Run)

1. **Konzept:** Ein-Satz-Logline → 3-Act-Outline
2. **Schreiben:** 25-30 Kapitel, je ~2.500 Wörter
3. **Lektorat:** Feedback, Korrekturen
4. **Cover:** Flux 2 Max Prompt generieren
5. **Export:** Als `buch-YYYY-MM-DD-[autor].md` speichern
